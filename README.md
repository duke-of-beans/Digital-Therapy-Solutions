# Digital Therapy Solutions
# Proper Sluice Property #1 — digitaltherapysolutions.com

## What This Is
An affiliate-driven online therapy recommendation site. Each page matches a visitor's
condition, insurance, or platform interest to the 3 best-fit platforms — with honest
tradeoff assessments, real pricing, and zero sponsored rankings.

## Design System
- **Fraunces** — hero headlines, nav brand, platform names, section headings, footer headings, reviewer name
- **Instrument Serif italic** — emotional accent moments (subheads, editorial dividers, pull quotes)
- **DM Sans** — body, pills, metadata, CTAs
- **DM Mono** — pricing data only
